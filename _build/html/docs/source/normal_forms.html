normal_forms.jet module¶

class normal_forms.jet.jet(f, x, k)[source]¶

Bases: object

truncated Taylor’s series

The jet is represented in both an expanded and closed form. The closed form is \(\sum_{0\leq d\leq k}\) coeff[deg]*basis[deg], where \(d\) is deg, while the expanded form is the list of numpy arrays coeff each with shape \((m,{n+d-1 \choose d})\) and the list of column vectors basis. coeff[deg][i,j] is the \(d^{th}\) derivative of \(f_i\) with respect to the \(j^{th}\) multiindex divided by the multiindex factorial and basis[deg] is of the form \(\begin{bmatrix}(x_1-x^{(1)})^d & (x_1-x^{(1)})^{d-1}(x_2-x^{(2)}) & \cdots& (x_n-x^{(n)})^{d} \end{bmatrix}^T\) where x \(=\begin{bmatrix} x^{(1)} &\cdots& x^{(n)}\end{bmatrix}^T\).

Parameters:
  • f (callable) – function that accepts n arguments and returns tuple of length m numbers, corresponding to mathematical function \(f:\mathbb{R}^n\rightarrow\mathbb{R}^m\)
  • x (number (if n==1) or iterable (if n>=1)) – center about which jet is expanded
  • k (int) – maximum degree of jet
m¶

int – number of output numbers, dimension of range of \(f\)

n¶

int – number of input arguments, dimension of domain of \(f\)

var¶

tuple of Sympy variables – x1, x2, …, xn representing input variables

coeff¶

tuple of numpy arrays – jet coefficients indexed as coeff[deg][coord,term] where \(0\leq\) deg \(\leq k\), \(0\leq\) coord \(\leq m\), and \(0\leq\) term \(<{m-1+d \choose d}\) and \(d\) is deg

basis¶

tuple of Sympy Matrices – monomial basis elements \(\prod_{i\in S}\) xi-x[i-1] where \(S\) is a set in the power set of \(\{1,\ldots,n\}\) and \(|S|\leq k\) indexed as basis[deg][term]

series¶

Sympy Matrix – Sympy Matrix with shape (m,1), symbolic representation of jet

multiindex factorial

Multiindex factorial is defined as \((\alpha_1,\ldots,\alpha_k)!=\alpha_1!\cdots \alpha_k!\). In this implementation, \(\alpha_i\) corresponds to the number of occurences of i in self.idx.

increment()[source]¶

Increment multiindex.

A ‘telephone-book’ ordering is used and within the multiindex indices are assumed to be increasing from left to right. For example, the multiindices with length n==2 and with indices less than idx_max=3 are, in increasing order: (0,0), (0,1), (0,2), (1,1), (1,2), (2,2). The next multiindex is cycled back to the first multiindex.
