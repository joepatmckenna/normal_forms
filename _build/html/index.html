Normal forms of dynamical systems in Python¶

This document describes how to use a Python package to compute normal forms of autonomous differential equations

\[\frac{dx}{dt} = f(x)\]

where \(f:\mathbb{R}^n\rightarrow\mathbb{R}^n\).

The first section of this documents presents some background and a few examples of normal forms, and the remaining sections present the Python classes that the main normal_forms.normal_form.normal_form class uses.
